# 📱 Mini CRM App (Flutter)

A simple and minimal Customer Relationship Management (CRM) mobile application built using Flutter.
This app allows users to manage customer details and call notes efficiently with a clean and user-friendly interface.

---

## 🚀 Features

* ➕ Add new customers
* 📋 View customer list
* 🔍 Search customers by name
* ✏️ Edit customer details
* 🗑️ Delete customers with confirmation
* 📝 Add notes for each customer
* 💾 Local data storage using Hive
* ⚡ Fast and lightweight performance

---

## 🛠️ Tech Stack

* **Flutter** – UI framework
* **Dart** – Programming language
* **Hive** – Local database
* **Provider** – State management

---

## 📂 Project Structure

lib/
│
├── models/ → Data models
├── providers/ → State management
├── services/ → Database logic
├── screens/ → UI screens
└── main.dart → Entry point

---

## 🧠 How It Works

* Customer data is stored locally using Hive database
* Provider is used to manage and update UI state efficiently
* The app supports full CRUD operations (Create, Read, Update, Delete)
* Search functionality filters customers in real-time

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/NileshKumar906/flutter-crm-app/
```

2. Navigate to project folder:

```
cd crm_app
```

3. Install dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```
